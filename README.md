# `@siddiqus/expressive-cli`

🏗 CLI for [@siddiqus/expressive](https://www.npmjs.com/package/@siddiqus/expressive) to bootstrap new projects.

## Installation

```bash
npm install -g @siddiqus/expressive-cli
```

## Usage
```bash
expressive-cli new appName

cd appName
yarn install
yarn start
```
